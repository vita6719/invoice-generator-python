📊 Invoice Automation System
Автоматизированная система генерации счетов с QR-кодами
🎯 О проекте
Этот инструмент предназначен для автоматизации малого бизнеса. Программа берет данные о клиенте (имя, услуга, сумма) и мгновенно превращает их в профессиональный PDF-счет с уникальным QR-кодом для оплаты.
💎 Почему это круто?
Полная автоматизация: Больше не нужно заполнять шаблоны в Word вручную.
QR-платежи: Ускоряет получение денег от клиента (сканируй и плати).
Кириллица: Полная поддержка русского языка (Arial).
Чистая архитектура: Код написан на классах (ООП), что ценится работодателями.
🚀 Быстрый старт для новичков
1. Установка окружения
Убедитесь, что у вас установлен Python. После этого установите три библиотеки, необходимые для работы графики и PDF:
bash
pip install reportlab qrcode pillow
Используйте код с осторожностью.

2. Запуск
Скопируйте код из файла main.py к себе.
Запустите его командой:
bash
python main.py
Используйте код с осторожностью.

Программа автоматически создаст папку /invoices и положит туда готовые документы.
📂 Структура проекта
main.py — Мозги системы. Логика генерации и обработки данных.
/invoices — Ваш цифровой архив. Сюда попадают все созданные счета.
temp_qr... — Временные файлы (удаляются автоматически после сборки PDF).
🛠 Технологии
Python 3.x
ReportLab — для сложной верстки PDF.
QRCode — для генерации платежных кодов.
