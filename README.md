# 📄 Автоматизированная система выставления счетов (Python)

Профессиональный инструмент для малого бизнеса и фрилансеров, позволяющий мгновенно генерировать PDF-счета с интегрированными QR-кодами для оплаты.

## 🚀 Ключевые возможности
*   **Мгновенная генерация PDF:** Создание официальных документов в один клик.
*   **Динамические QR-коды:** Автоматическое встраивание платежных ссылок в тело счета.
*   **Локальное хранение:** Программа автоматически организует архив счетов в папке `/invoices`.
*   **Готовое решение (EXE):** Доступна скомпилированная версия для Windows, не требующая установки Python.

## 🛠 Технологический стек
*   **Python 3.x** — основная логика.
*   **ReportLab** — верстка и генерация PDF-документов.
*   **QRCode & Pillow** — создание и обработка графических кодов.
*   **PyInstaller** — сборка автономного приложения.

## 💻 Инструкция по запуску

### Для пользователей (Windows):
1. Перейдите в раздел **Releases** справа.
2. Скачайте файл `InvoiceGenerator.exe`.
3. Запустите файл — счета появятся в созданной папке автоматически.

### Для разработчиков:
1. Клонируйте репозиторий.
2. Установите зависимости: 
   `pip install reportlab qrcode pillow`
3. Запустите `main.py`.
